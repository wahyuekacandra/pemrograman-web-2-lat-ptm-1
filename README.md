# LAT PTM 1 - Pemrograman Web 2

Repositori ini berisi latihan praktikum (LAT) PTM 1 untuk mata kuliah **Pemrograman Web 2**.

## Isi
- `latihan1.php` - Contoh variabel global vs lokal
- `latihan2.php` - Contoh penggunaan `global` dalam fungsi
- `latihan3.php` - Hitung luas lingkaran dengan konstanta
- `latihan4.php` - Contoh integrasi PHP dalam HTML dan operasi sederhana

## Cara Menjalankan
Buka file-file `.php` di web server (misalnya XAMPP) dan akses melalui browser.
